Info About Course
In order to add a faculty member and or semester/year use insert schedule. The current database design includes courses that are supplied by the institution and schedules of year semester and instructor.
This is an excerpt from the documentation :
"Currently, classes could have the same prefix, number, section, and assignment. This is
due to classes generally having the same expectations throughout years of education. The class
entity should be separate from a class object entity which would include a year, semester, and
teacher. In an educational system, a class will be planned and scheduled before a guaranteed time
and semester is placed. Therefore, having this information in the same entity is creating a
dependency of information. Also, anytime information is being heavily repeated, there is high
probability of mistakes. It is a good rule of thumb to separate attributes that are closest associated
with a concept and idea."
